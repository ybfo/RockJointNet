GeoSPIN framework: stress-path integral view
============================================

Inputs
------

  static joint context c
  = [JRC, JCS/UCS, phi_b, specimen length, Young's modulus, ...]

  queried normal stress sigma_n


Step 1. Convert normal stress to a dimensionless path coordinate
----------------------------------------------------------------

  sigma_n  --->  s = log(1 + sigma_n / sigma_ref)

  sigma_ref is computed from the training partition only.
  The model predicts the accumulated strength along the path from 0 to s.


Step 2. Encode only static context
----------------------------------

  c  --->  context encoder E_phi  --->  z_c

  z_c is a context representation.
  Sigma-dependent quantities are not treated as fixed encoder inputs.


Step 3. Build quadrature nodes along the stress path
----------------------------------------------------

  target path length: s

  midpoint nodes:

      u_k = 0.5 * s * (x_k + 1),        k = 1, ..., K

  quadrature weights:

      omega_k


Step 4. Compute online physics features at each node
----------------------------------------------------

  for each node u_k:

      sigma_k = sigma_ref * (exp(u_k) - 1)

      q_k = q(sigma_k, c)

  q_k may include stress-dependent physics quantities such as
  sigma_k / JCS, log(JCS / sigma_k), trusted Barton-Bandis strength,
  and trust-region weights.


Step 5. Predict a nonnegative strength rate
-------------------------------------------

  rate network:

      a_k = C_psi(u_k, z_c, q_k)

  positive rate:

      r_k = softplus(a_k) + epsilon

  This makes d tau_hat / d s strictly nonnegative.


Step 6. Integrate the positive rate
-----------------------------------

  predicted peak shear strength:

      tau_hat_p(s, c) = (s / 2) * sum_{k=1}^{K} omega_k * r_k


Hard physical guarantees
------------------------

  zero-stress boundary:

      tau_hat_p(0, c) = 0

  fixed-context monotonicity:

      d tau_hat_p / d sigma_n >= 0, when c is fixed

  nonnegative prediction:

      tau_hat_p(s, c) >= 0


Training signals
----------------

  data loss:

      Huber(asinh(tau_hat_p), asinh(tau_p))

  trusted-region empirical prior:

      Barton-Bandis regularization is used only where its trust weight is active.

  optional curve head for local direct-shear records:

      shear-displacement and deformation-proxy curves are reconstructed only
      for local monotone direct-shear data, not claimed for cyclic histories.


One-line visual summary
-----------------------

  c -------------------> E_phi(c) = z_c ---------------------------+
                                                                  |
  sigma_n ---> s ---> quadrature nodes u_k ---> online q_k --------+--> C_psi
                                                                        |
                                                                        v
                                                             r_k = softplus(.) + eps
                                                                        |
                                                                        v
                                      tau_hat_p = integral_0^s r(u, c, q(u,c)) du
                                                                        |
                                                   hard boundary + monotonicity
